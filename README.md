ipws
====

Integrated Perl Web Services, bringing an awesome forum, wiki, blog etc. software to Perl's ecosystem

Dependencies
------------
- Mojolicious
      - Mojolicious::Plugin::YamlConfig
      - Mojolicious::Plugin::Database
- Rose::DB::Object
- CryptX
- Math::Round
- Text::Password::Pronounceable
- Try::Tiny
- Module::Find
- Guard
